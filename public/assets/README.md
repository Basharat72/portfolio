# Assets

Drop your personal files in this folder — the site references them by these exact names:

| File | Used for |
|---|---|
| `profile.jpg` | Hero portrait (a square-ish photo ≥ 700×820px works best). Until it exists, the site shows a stylish "BA" monogram instead. |
| `Basharat-Mubashir-Ahmed-CV.pdf` | The "Download CV" button in the hero. |

Tips:
- Compress the photo before adding it (e.g. https://squoosh.app) — aim for under 150 KB.
- If you rename either file, update the matching path in `index.html`.
